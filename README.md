## 📖 Overview

This project is a **Nestlé-inspired Product Showcase Website** designed to demonstrate modern front-end development using **HTML5** and **CSS3** without any frameworks or JavaScript.

The website presents products across multiple categories with engaging hover effects, smooth animations, stylish layouts, and a clean user interface.

It serves as a beginner-to-intermediate front-end project focusing on:

- Modern webpage design
- CSS animations
- Flexbox layouts
- Interactive product cards
- UI/UX fundamentals
